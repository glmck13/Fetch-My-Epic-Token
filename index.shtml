Welcome to Fetch My Epic Token!

This page enables anyone with an Epic "MyChart" account to obtain an access token and patient ID so they can pull information from their Electronic Health Record (EHR) using the FHIR API. If you've never heard of the FHIR API and don't know how to use it, check out the examples on the HowTo page. Afterwards, if you're feeling a little more adventurous take a look at getEHR.cgi on my Github site. This script uses the FHIR API to extract all of the lab results, test reports, and clinical notes in your EHR, and packages these in a ZIP file for download. A copy of the script is installed on this site for you to use. To run it, click the button that appears just below this intro after you're logged in.

To get started, select your Epic/MyChart provider from the list below and click Login. Enjoy!
